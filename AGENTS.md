# AGENTS.md

## Knowledge Notes

- このリポジトリでは、調査内容や構造理解などの Markdown メモを `docs/agent-notes/` に蓄積する。
- 作業を始める前に、必要に応じて `docs/agent-notes/` 配下の既存メモを確認し、過去の整理内容を参照する。
- 新しく把握した構造、設計判断、データモデル、調査結果は、再利用価値がある場合に `docs/agent-notes/` 配下へ Markdown として保存する。
- 既存メモと内容が重なる場合は、重複ファイルを増やしすぎず、追記または関連ファイルの追加で整理する。
